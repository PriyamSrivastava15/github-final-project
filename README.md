# Simple Interest Calculator

## Purpose
This project provides a simple script to calculate simple interest, supporting our micro-finance startup's mission to empower low-income individuals with accessible financial tools. This repository marks the first step in migrating our codebase from Subversion (SVN) to Git/GitHub.

## What is Simple Interest?
Simple interest is calculated using the formula:

SI = (P x R x T) / 100

Where:
- P = Principal amount
- R = Rate of interest (per annum)
- T = Time period (in years)

## Getting Started
1. Clone the repository:
```git clone https://github.com/<your-username>/simple-interest-calculator.git```
2. Run the script (adjust based on your language):
```python simple_interest.py```
